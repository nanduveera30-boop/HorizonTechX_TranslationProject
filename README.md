# 🌍 AI Voice Translation Assistant

An AI-powered multilingual voice translation application built using Python, Streamlit, Speech Recognition, Text-to-Speech, and Google Translate API.

## 🚀 Features

* 🎤 Voice Input
* 🌐 Multi-language Translation
* 🔊 Text-to-Speech Output
* 📝 Manual Text Translation
* 📚 Translation History
* 📊 Character Counter
* 💾 Download Translations
* 🎨 Interactive Streamlit UI

## 🛠️ Technologies Used

* Python
* Streamlit
* Google Translate API
* SpeechRecognition
* Pyttsx3
* PyAudio

## 📂 Project Structure

```text
Translation_Project
│
├── app.py

```

## ⚙️ Installation

```bash
git clone https://github.com/nanduveera30-boop/HorizonTechX_TranslationProject.git
cd HorizonTechX_TranslationProject
pip install -r requirements.txt
streamlit run app.py
```

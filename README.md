# 🌍 Smart Translator Pro

A professional AI-powered language translation platform built using Python, Streamlit, Deep Translator, and Browser Voice Recognition.

Smart Translator Pro enables users to translate text across multiple languages using either manual text input or browser-based voice input, providing a seamless multilingual communication experience.

---

## 🚀 Features

### 🌐 Multi-Language Translation

Translate text between multiple global languages instantly.

### 🎤 Voice Input Support

Use browser-based voice recognition to convert speech into text.

### ✍️ Manual Text Translation

Type text directly and translate it into your desired language.

### 📊 Translation Analytics

Track:

* Total translations
* Characters processed
* Word count
* Translation history

### 📥 Download Results

Download translated text as a text file.

### 🕘 Translation History

View previously translated content within the session.

### ☁️ Cloud Deployment Ready

Compatible with Streamlit Community Cloud.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/nanduveera30-boop/HorizonTechX_TranslationProject.git
```

Navigate to the project directory:

```bash
cd HorizonTechX_TranslationProject
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
